# Semantic Search Engine using ElasticSearch & FAISS

A semantic information retrieval system combining traditional keyword search with modern embedding-based vector retrieval.

This project explores two approaches to document retrieval:

- Phase A: Sparse retrieval using ElasticSearch and BM25
- Phase B: Dense semantic retrieval using Sentence Transformers and FAISS

The system supports indexing, searching, ranking, and semantic similarity matching on textual documents.

## Project Overview

### Phase A — ElasticSearch Retrieval

Implemented a keyword-based search engine using ElasticSearch.

Features:
- CSV document preprocessing
- Custom ElasticSearch index creation
- BM25 ranking
- Query execution
- Top-k result retrieval

### Phase B — Semantic Search with FAISS

Implemented a semantic search pipeline using dense embeddings.

Features:
- SentenceTransformer embeddings
- Vector indexing with FAISS
- Cosine similarity search
- Semantic retrieval
- Dense ranking pipeline
